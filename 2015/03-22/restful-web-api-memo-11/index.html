Chapter 10. The Hypermedia Zoo (Part. 2)

  • アプリケーションセマンティクスカタログ

IANA Registry (Link Relation)

  • media-type
    • 分け隔てなくすべて
  • セマンティクス
    • 一般的なナビゲーション
  • あらゆる表現に対して使用できる
  • 各リンクリレーションには、human-readableなドキュメントと、定義元へのリンクが与えられている

Microformats Wiki

  • media-type
    • HTML

  • セマンティクス

    • オンラインで検索を行うことに関するもの

  • hCalendar

    • 時間にまつわるイベントを記録
    • iCalendarベース (RFC2445)

  • hCard

    • 人と組織について記述
    • vCardベース (RFC2426)
    • 7章参照

  • XFN

    • 人との関係を記述するリンクリレーションのセット

  • XOXO

    • ドキュメントの要点を記述
    • 新たなアプリケーションセマンティクスは定義していない
    • htmlの範囲内でやりくりしている

  • adr

    • hCardのサブセット
    • 住所のみを取り扱う

  • geo

    • hCardのサブセット
    • WGS84基準による緯度と経度

  • hAtom

    • ブログ投稿
    • Atomフィードベース(RFC4287)

  • hListing

    • 雇用や広告に関するサービスを一覧
    • hCard, hCalendar, hReviewを組み合わせたもの

  • hMedia

    • がぞう、ビデオ、オーディオについての基本的なメタデータ

  • hNews

    • hAtomの拡張
    • ニュース記事を追加する

  • hProduct

    • 製品リスト

  • hRecipe

    • レシピ

  • hResume

    • Resume/CVs ???

  • hReview

    • レーティング付きレビュー

  • html5由来のリンクリレーション

    • author, nogollow, tag, license

Schema.org

  • microdataの総本山
  • microformats wikiと類似したアプリケーションセマンティクスが多く登録されている
  • 主に利用者視点

Dublin Core

  • (うまく纏めれなかった・・・)

Activity Streams

  • Activityと呼ばれるオンライン活動(ビデオをみる、ブログを投稿する) を表現する標準
    • Activity
      • オンラインを通して対話することに関するアプリケーションセマンティクス
        • Article, Event, Group, Person,…
    • Atomフィードとして表現する
    • JSON版もある
      • ハイパーメディアコントロールではない
  • Actor, Verb, Objectで構成される
    • Actor
      • たいてい利用者
    • Verb
      • Actorが何か行うこと
      • 不安定な状態遷移名
        • join, follow, cancel
    • Object
      • ActorがなにかVerbする対象
  • schema.orgのmicrodataに多くの重複が見受けられる
