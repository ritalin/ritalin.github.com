（みんなには内緒だよ）

Hackintosh (Hackロム)に頼ること無く、インストールできた 真っクロに限りなく近いクロ 見えなくなった時は、お察しください。

前提 VirtualBoxインストール済みv 4.2

App StoreからMountain Lionを購入。 インストーラの１画面目→即終了 ディスクイメージ抽出

ディスクユーティリティを起動 「変換」選択 変換もと：抽出したの 変換先：フォーマットをDVD / CDマスター 変換待つ・・・

VBで仮想マシン追加 タイプ：Mac OS X バージョン：Mac OS X (64 bit) 以降は通常通りすすめて作成する

設定を開き ストレージのCD /DVDに変換したイメージを指定する。

VMを実行すると、インストーラから起動する（結構時間かかる） VM内のディスクユーティリティを起動し、パーティションをフォーマットする（不要かもしれない） Mac OS 拡張（ジャーナリング）、GUIDパーティションテーブル

ディスクユーティリティ終了→インストールアプリ起動 設問に従ってポチポチ

インストール始まる。

最後までいくと再起動 完了かと思いきや、インストールがおこなわれてる。 ２ループ目は不明だったので、静観してたけど、３ループ目も同じように進もうとしたため、VMを強制停止（ウインドウ閉じればアラートでる）

たぶんリカバリインストールが始まってたぽい気がする。

VBの設定開き CD / DVDをアンマウントする

その状態でVMを実行すると、インストールプロセスの続きが実行された。 言語とかネットワーク設定とか聞いてきたような気がするけど、必須入力以外はすべてスキップした

VBの関係上なのか、英語版にされる。

起動の裏側が見れる ２本指スクロールが、Snow Leopard基準。つまり本来のMountain Lと逆。 聞かないと思ってた

放置しておくと、結構フリーズ状態になるので要注意

小ネタのつもりが原爆級になったかもしれないけど仕方ないね。
